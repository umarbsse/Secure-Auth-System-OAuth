# OAuth Login System - Google & Facebook

A simple OAuth implementation supporting both Google and Facebook login using plain PHP.

## What's Inside

### Google Folder
Contains files for Google OAuth login functionality.

### Facebook Folder
Contains files for Facebook OAuth login functionality.

## Quick Start

1. Set up your OAuth credentials (Google Cloud Console / Facebook Developers)
2. Configure the credentials in the respective config.php files
3. Run `php -S localhost:8000`
4. Open the index.php from either Google or FaceBook folder
